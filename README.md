# ABCDrado hehe

Recriação de uma composição concretista usando p5.js. O projeto renderiza uma grade de cubos onde cada face lateral pode ter sua cor alterada ao clicar na tela, ciclando por uma paleta de cores predefinida.

## Como rodar

Abra o arquivo `index.html` e rode o Live Server diretamente no navegador.

## Interação

Clique na tela para trocar a cor das faces laterais dos cubos.


## OBS

A imagem reproduzida neste projeto não possui autoria definida. Ela é amplamente utilizada como referência quando o assunto é concretismo.
